# RPG Правила

## Концепция

Ты — персонаж. Каждая выполненная задача даёт опыт в определённый навык.
Опыт накапливается, навыки растут, персонаж повышает уровень.

## Навыки

| Навык | Тег на задачах | Описание |
|---|---|---|
| Python | `#skill/python` | Скрипты, боты, парсеры, бэкенд |
| SQL | `#skill/sql` | Запросы, схемы, оптимизация |
| Linux | `#skill/linux` | Сервер, bash, systemd, сети |
| Docker | `#skill/docker` | Контейнеры, compose, registry |
| VBA | `#skill/vba` | Excel, Access, макросы |
| AI | `#skill/ai` | Ollama, промпты, RAG, embeddings |

| Learning | `#skill/learning` | Курсы, книги, исследования |

## Как обновлять (еженедельно)

1. Открываешь Weekly Note.
2. Смотришь таблицу XP за прошедшие дни (из Daily Notes).
3. Открываешь нужные заметки навыков (`02 Profile/Skills/`).
4. Меняешь `xp:` на новое значение.
5. Если xp перевалил порог уровня — меняешь `level:`.
6. Открываешь `Персонаж.md`, обновляешь `xp_total:` и `level:`.
7. Карточка перерисуется автоматически.

## Как ставить теги на задачи

Каждая задача должна иметь тег навыка:
- Написать парсер для API #skill/python 📅 2026-05-02
- Настроить nginx на сервере #skill/linux 📅 2026-05-03
- Пробежка 5 км #skill/fitness 📅 2026-05-02

Одна задача может иметь несколько тегов навыков — тогда XP делится между ними.

## Достижения

Создаёшь вручную в `02 Profile/Achievements/` по шаблону:

```yaml
***
type: achievement
date: 2026-05-02
reward: +50 XP
skill: python
***
```

Примеры достижений:
- Первый запущенный сервер
- 10 задач подряд без пропуска
- Завершён первый квест
- 100 XP в одном навыке
