**Ключевые поля Products:**
- `ExtNomCode TEXT100` — (не `ExtNomCodeOverride`, старое название убрано)

- `Quantity DOUBLE NOT NULL DEFAULT 1`

- `IsOnSchedule YESNO`
- `OwnershipStatus TEXT30`, `WriteOffStatus TEXT30`
- Металлы: `GoldGrams`, `SilverGrams`, `PlatinumGrams`, `MPGGrams`

## Заметки по архитектуре

- DDL актуален в `ModCreateDB.GetDDLStatements`. `db_schema_vvt_v7.md` нужно синхронизировать с ним.
- `Products` — центральная таблица, 40+ полей, split на 4 строки sqlA/B/C/D для обхода ограничения VBA на длину строки.
- Все FK на `Users.UserID` в критичных таблицах есть (AuditLog, ChangeRequests, Products и т.д.).
- `ChangeRequests.LockToken TEXT100` — механизм оптимистичной блокировки при редактировании.
- `UserSessions.WorkbookHost TEXT100` — для отслеживания конкретного Excel-экземпляра.
- `ADDINPATH` — путь к `EncryptVVT.xlam`, берётся из ModSettings через `GetConstant`.
