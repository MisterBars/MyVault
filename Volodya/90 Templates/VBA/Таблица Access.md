# Таблица `TableName`

## Назначение
Таблица хранит `краткое описание сущности`.

Используется для `описание роли таблицы в системе`.

## Бизнес-смысл
Одна запись = `что именно означает одна строка таблицы`.

Таблица описывает `реальную сущность / объект / событие / связь`.
