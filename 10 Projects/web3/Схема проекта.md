# 🗺️ Схема проекта web3

> Фаза 3 учебного плана — ~4 месяца, ~22 модуля + 2 пет-проекта.
> Связана с: [[web3]]

---

## 🌐 Нулевой блок — Основы интернета (доп.)

> Добавлен для общего понимания, перед погружением в Web3.

| # | Тема | Цель |
|---|------|------|
| 0.1 | Что такое Web (Web 1.0) | Статичный интернет: HTML-страницы, только чтение, нет интерактивности |
| 0.2 | Что такое Web 2.0 | Динамичный интернет: пользователи создают контент, централизованные платформы (Google, Facebook), REST API, AJAX |
| 0.3 | Проблемы Web 2.0 | Централизация данных, цензура, монополии, утечки данных, зависимость от платформ |
| 0.4 | Что такое Web3 | Децентрализованный интернет: блокчейн + смарт-контракты + токены + децентрализованные хранилища. Пользователь владеет данными |
| 0.5 | Отличия Web2 vs Web3 | Сравнение архитектур: сервер vs нода, БД vs блокчейн, логин/пароль vs кошелёк, API vs смарт-контракт |

## 🎯 Пет-проекты

### Пет-проект 1 — On-chain аналитика
```
Ethereum → ethers.js → события → PostgreSQL → REST API (Fastify/TypeScript) → React-дашборд
```
- Читаем события смарт-контрактов из Ethereum (или тестовой сети)
- Сохраняем в PostgreSQL
- Строим REST API
- Визуализируем в React

### Пет-проект 2 — Web3 DApp (ERC-721 + IPFS)
```
Solidity (ERC-721) → Hardhat → Sepolia → React + Wagmi → IPFS/Pinata (метаданные)
```
- Пишем и деплоим NFT-смарт-контракт
- Загружаем метаданные в IPFS через Pinata
- React-интерфейс с подключением MetaMask
