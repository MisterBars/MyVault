# 0.6 gitignore и gitattributes

> Модуль про чистоту репозитория: какие файлы не должны попадать в Git и как задавать правила для текстовых файлов.

## Цель модуля

- Научиться исключать мусор и служебные файлы из Git.
- Понять, зачем нужен `.gitattributes`.
- Подготовить хороший шаблон под будущие Python-проекты.

## Теория


### Ключевые концепции

- Концепция 1 — описание
- Концепция 2 — описание

### Ключевые идеи

- `.gitignore` скрывает лишние файлы от отслеживания.
- В Git не нужно коммитить виртуальные окружения, кеши, временные файлы и секреты.
- `.gitattributes` помогает управлять поведением Git для файлов, включая окончания строк.

> Основная идея одним абзацем — своими словами, без копипаста.
