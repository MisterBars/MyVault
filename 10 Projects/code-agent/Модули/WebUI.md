# Модуль WebUI

## Назначение
Веб-интерфейс проекта.

Нужен для:
- общения с агентом
- хранения и просмотра бесед
- отображения planner/worker шагов
- работы с проектами
- загрузки файлов для RAG

## Важные решения
- UI развивается параллельно с backend
- нужно показывать не только финальный ответ, но и ход рассуждения по ролям
- интерфейс должен работать с проектами и сессиями
- загрузка файлов должна быть частью UI

## Основная ответственность
- чат
- список бесед
- выбор проекта
- отображение ролей planner/worker
- загрузка файлов
