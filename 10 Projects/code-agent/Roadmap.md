# Roadmap

## Phase 1 — Базовая multi-agent логика
- PlannerAgent
- WorkerAgent
- Orchestrator
- текстовые ответы
- возврат подзадачи на перепланирование

## Phase 2 — Web UI
- чат-интерфейс
- отображение сообщений по ролям
- список бесед
- хранение истории
- выбор проекта / сессии

## Phase 3 — RAG
- ручная загрузка файлов
- индексация документов
- retrieval для planner и worker

- проектная база знаний

- анализ структуры проекта
- работа с git-репозиториями

## Phase 5 — Web
- просмотр веб-страниц
- извлечение полезной информации
- сохранение нужного контекста в RAG

## Phase 6 — Project memory
- отдельные проектные беседы
- общая память
- проектная память
- переиспользование контекста между сессиями
